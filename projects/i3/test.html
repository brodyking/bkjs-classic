    #Introduction

    This project is a guide on how to get your i3-gaps to look like mine. This guide expects you to be using Ubuntu 21.10 with GNOME installed.

    #Installing i3-gaps

    Installing i3 gaps is a very easy process. Simply type the following into the terminal.

    sudo add-apt-repository ppa:regolith-linux/release
    sudo apt update
    sudo apt install i3-gaps

    After that, logout and when logging back in, pick i3 as your DE.

    #Using i3-gaps

    i3-gaps is a fork of i3. So the best place to look for documentation is to look online for i3 tutorials. There are many great resources out there to learn i3. Here are some basic commands to get you started.

    Mod + Enter = Creates a terminal window,
    Mod + D = Opens dmenu, a program launcher made by suckless,
    Mod + Shift + Q = Close the current window,
    Mod + Arrow Keys = Move to window next to currently selected window,
    Mod + Number Keys = Move to a different workspace,
    Mod + Shift + Number Keys = Move currently selected window to a different workspace,
    Mod + Shift + Arrow Keys = Move currently selected window around,

    #Ricing

    Ricing is when you customize your window manager or DE. We will install a few things to make our i3 look better.

    #Bumblee Status

    Bumblebee status is a program that replaces the status bar at the bottom right. We will need pip (pythons package manager) to install it. Type this into the terminal to install pip.

    sudo apt-get install python3-pip

    After this, we can install bumblebee-status.

    sudo pip install --user bumblebee-status

    Run the command again and it should tell you where it is stored. Write this down for later. Lets get to setting it up.

    Head into the i3 config with this command.

    nano ~/.config/i3/config

    Press Ctrl + Shift + Minus , then Ctrl + V. This will bring us to the bottom of the file. You should see a element that looks somthing like this:

    bar {
    status_command i3status
    }

    We want to change this to be our bumblebee-status bar. Set everything to how it is in this screenshot, including the tabbing. Remember to replace my bumblebee install path with your own.
